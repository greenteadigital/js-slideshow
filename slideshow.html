js-slideshow

A javascript slideshow for images

Features:
-Auto-start on page load
-User controlled; play, pause, skip forward/back
-Continuous looping, fade transitions, auto-pause on skip
-Dynamically links to larger image or external site (please click through the images)
-Remembers position w/ cookies; user doesn't start over upon returning to page
-Support for multiple sizes/formats
-Doesn't hinder page load time, most images load in background after page has rendered
-Smart buttons; images/tooltips change to fit context
